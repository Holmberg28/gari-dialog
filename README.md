# GariDialog

GariDialog is a lightweight, customizable, zero-dependency dialog & prompt library for web applications.  
It offers clean UI dialogs with optional icons, dynamic inputs, button actions, animations, and promise-based results.

Perfect for confirmations, alerts, prompts, or custom interactive dialogs.

**See live demo:** [holmberg28.github.io/gari-dialog](https://holmberg28.github.io/gari-dialog/index.html)

---
## üöÄ Features

- [NEW] Sound Notifications!
- Clean, modern dialog UI 
- Supports alerts, confirms, prompts  
- Add inputs with labels (text, number, email, password, textarea)  
- Custom icons (FontAwesome, system emoji, or others)  
- Custom colors  
- Smooth zoom-in/zoom-out animations  
- Promise-based API (`GariDialog().then(...)`)  
- Auto-focus on first button
